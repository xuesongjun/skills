# 技能：Refactor_Pro (重构助手)

## 触发词
- 优化这段代码、重构、审查代码

## 执行逻辑
1. **复杂度分析**：识别圈复杂度过高、嵌套过深的函数。
2. **提出建议**：给出重构方案（如：提取函数、合并条件分支、引入策略模式）。
3. **安全实施**：
   - 在重构前，询问是否有现成测试。
   - 重构过程保持外部接口（API）不变，确保向后兼容。
4. **性能自检**：检查是否有不必要的循环或内存泄漏风险。
