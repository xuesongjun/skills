# 技能：Test_Master (测试专家)

## 触发词
- 写测试、增加覆盖率、测试这个函数

## 执行步骤
1. **分析环境**：识别项目使用的测试框架（Jest, Vitest, Pytest 等）。
2. **识别边界**：针对目标代码，列出：
   - 正常路径 (Happy Path)
   - 边界条件 (Edge Cases)
   - 异常路径 (Error Handling)
3. **编写测试**：
   - 使用 Mock 掉外部依赖（API、数据库）。
   - 确保测试描述清晰（Given-When-Then 结构）。
4. **运行验证**：尝试运行测试并根据报错自动修复代码。
