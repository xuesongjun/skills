# Git 提交

## 操作步骤

1. 运行 `git status` 查看变更文件
2. 运行 `git diff` 查看具体改动
3. 运行 `git log --oneline -5` 查看最近提交风格
4. 分析变更，生成中文提交信息
5. 使用 `git add` 添加相关文件（不要用 `git add -A`）
6. 执行 `git commit`

## 提交信息规范

本项目使用中文提交信息，格式参考：

```
<动作><功能模块描述>
```

示例：
- `优化寄存器对比`
- `对比寄存器时，增加排除寄存器功能`
- `优化寄存器对比速度`
- `新增寄存器对功能`
- `新增log界面`

## 注意事项

- 提交信息使用中文
- 不要提交 `venv/` 目录
- 不要提交 `.env` 或包含敏感信息的文件
- 不要提交 `__pycache__/` 目录
